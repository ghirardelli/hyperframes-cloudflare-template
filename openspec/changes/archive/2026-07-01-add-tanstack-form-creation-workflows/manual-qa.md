## Manual QA Pass

Use this checklist after starting the local app:

- Log in with empty fields, invalid credentials, and valid credentials.
- Create an admin user with an existing organization and with a new organization.
- Update profile name, change password, and verify password fields clear after success.
- Edit, cancel, and save project metadata from `/projects`.
- Generate from the main Manual Prompt tab with duration and gallery context.
- Render from the Render tab with each resolution and format option.
- Start a website-to-video workflow from the AI Agent, then open the workflow wizard.
- Select wizard stages, edit an editable markdown artifact, discard changes, and save changes.
- Ask the wizard-scoped agent to inspect a stage, propose a patch, apply an approved patch, and rerun validation.
- Confirm Studio still opens through the existing project Studio route.
